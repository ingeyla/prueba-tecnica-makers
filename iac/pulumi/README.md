# Pulumi (Bonus)

Implementacion alternativa para quienes prefieren Pulumi sobre Terraform.

## Objetivo

- Demostrar modelado de recursos cloud en TypeScript.
- Identificar y corregir decisiones inseguras o no operables.

## Uso (referencial)

```bash
cd iac/pulumi
npm install
pulumi up
```

No es obligatorio ejecutar. Se evalua criterio tecnico y calidad de propuesta.
